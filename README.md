## Installation 

1. - Clone the repo
 ```sh
git clone https://github.com/ggab13/memory-game.git
   ```

2. - Navigate to the project directory
```sh
 cd memory-game  
 ```
3. - Install dependencies
 ```sh
npm install
```
4. - Navigate to dist folder
 ```sh
 cd memory-game  
 ```
5. - Start the app
  ```sh
npm start
 ```
